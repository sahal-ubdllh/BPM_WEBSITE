Cara Pakai Dashboard:
  • Semua aspirasi yang masuk akan otomatis muncul di Google Sheets di bawah
  • Klik tombol "Buka di Tab Baru" untuk edit lebih leluasa
  • Export CSV: Klik "Download CSV" atau di Google Sheets pilih File → Download → CSV
  • Update status: Edit langsung kolom "Status" di Google Sheets
  • Data tersimpan otomatis oleh Google (backup gratis!)
